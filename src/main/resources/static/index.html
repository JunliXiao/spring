練習透過 Spring Boot 開發 REST API 端點

  • 取得所有使用者：GET /api/v1/users
  • 按 username 取得使用者：GET /api/v1/users/{username}
  • 按 username 更新使用者：PUT /api/v1/users/{username}
  • 按 username 刪除使用者：DELETE /api/v1/users/{username}
  • 按 request body 內容新增使用者：POST /api/v1/users
